This project has not only given me more experience with Python, but has also greatly deepened my understanding of Git.
HVC(Hayden's Version Control) is a simple Git implementation that utilizes Python in order to facilitate version control functionality.


## Functionality
- Initializing a repository
- An index/stage for tracking files
- Converting files to compressed, unique hashed objects
- Committing changes to history
- Full logging functionality, such as providing log output, and logging branch and HEAD changes
- Creating, managing, switching, and deleting branches
- Viewing object content, type and size
- A .hvc_ignore file to prevent files/folders from being tracked
- Merge (Coming Soon)
